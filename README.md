# notes-react
Membangun Aplikasi Catatan Menggunakan React

## Usage
- Copy this Repo;
  ```
  git clone https://github.com/Id-Yuu/notes-react.git
  ```

- install with `bun` or `npm`
  ```
  > with bun
  bun install
  
  > with npm
  npm install
  ```

- develop local
  ```
  > with bun
  bun run dev

  > with npm
  npm run dev
  ```
